为您的 CI 管道|线束提供 3 个插件

原文：https://www.harness.io/blog/3-plugins-to-consider-for-your-ci-pipeline#0001-01-01

在本帖中，我们将分享你应该考虑集成到你的持续集成(CI)管道中的 3 个无人机插件。

用于 CI/CD 的平台 Harness 为您的软件交付生命周期提供了企业级特性。我们很高兴地欢迎 Drone，一个持续集成(CI)平台加入 Harness 家族，以扩展 Harness CI 功能。在以前的帖子中，我们已经分享了如何开始使用无人机。然而，在绿地或棕地环境中，在竞争情报平台上创建一个强大的竞争情报管道可能是一个挑战。在本帖中，我们将分享 3 个你应该考虑集成到你的持续集成管道中的无人机插件。

第一名:Sonarqube:

CI 管道是质量工程和保证的支柱，因此 CI 流程应该自动化，并且将测试作为基石。
