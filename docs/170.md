## 云成本管理-将您的过滤器和组保存为透视图

云成本管理非常适合对云成本数据进行切片和切块——寻找所有可以优化云支出的方法。到目前为止，每次使用 CE 时，您都必须执行相同的过滤和分组。随着[驾驭](https://harness.io/blog/gain-new-perspective-cloud-bill/)视角的释放，辛劳已被永久移除。透视图允许您创建一组过滤器和分组，安排报告，并保存它们以备将来需要时使用。每个用户都可以创建许多透视图，并与其他用户共享。你会喜欢这段从线束中保存新功能的时间。

## AWS 市场现已提供云成本管理

当软件在 AWS 市场上可用时，它使得购买过程变得更加容易。现在，您可以利用贵公司预先承诺的 AWS 支出来购买 Harness 云成本管理。没有必要经历申请新预算的麻烦。只需[通过 AWS marketplace](https://aws.amazon.com/marketplace/pp/B08DRZ3JLZ?ref_=srh_res_product_title) 启动购买流程，正确的结算就会自动启动。掌控您的云账单从未如此简单。

## GCP 工作负载身份支持

Harness 用户现在能够使用 Harness 委托和工作负载身份来认证 GCP 服务，比如从 GCS 和 Google cloud build 访问工件。用户现在可以在创建 GCP 云提供商时选择特定的代表，而不是提供服务帐户密钥。

这使得管理密钥轮换变得简单，因为用户只需要在带有工作负载标识设置的 GKE 实例中安装代理。

## 通过 API 创建 HashiCorp Vault Secret Manager

用户现在可以通过 API 创建 HashiCorp 保险库机密管理器。这将自动创建 HashiCorp Vault Secrets 管理器。用户还可以通过 API 将已配置的机密管理器的范围扩大到特定的应用程序和环境，任何添加的底层机密都将继承该范围。
