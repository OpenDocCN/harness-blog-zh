  • 自动关闭空闲的虚拟机和容器，并在现场实例上无中断地动态运行它们，从而将成本降低高达 75%。

新功能

CD -将事件发布到 HTTP 端点

用户现在可以从他们选择的任何日志工具获得关于他们的部署管道的事件通知。他们可以设置事件，当管道开始、完成或者在执行过程中暂停时，将这些事件发送到任何 webhook 端点。这将有助于用户更好地了解整个开发运维流程。这些事件也可以用 GraphQL APIs 来管理。

用于批准的 CD - API 支持

Harness 用户现在可以轻松批准或拒绝他们的 API 部署管道。这将有助于自动化审批流程，并缩短总体交付时间。用户不必访问 UI 来手动批准/拒绝管道。
