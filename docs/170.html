云成本管理-将您的过滤器和组保存为透视图

云成本管理非常适合对云成本数据进行切片和切块——寻找所有可以优化云支出的方法。到目前为止，每次使用 CE 时，您都必须执行相同的过滤和分组。随着驾驭视角的释放，辛劳已被永久移除。透视图允许您创建一组过滤器和分组，安排报告，并保存它们以备将来需要时使用。每个用户都可以创建许多透视图，并与其他用户共享。你会喜欢这段从线束中保存新功能的时间。

AWS 市场现已提供云成本管理

当软件在 AWS 市场上可用时，它使得购买过程变得更加容易。现在，您可以利用贵公司预先承诺的 AWS 支出来购买 Harness 云成本管理。没有必要经历申请新预算的麻烦。只需通过 AWS marketplace 启动购买流程，正确的结算就会自动启动。掌控您的云账单从未如此简单。
