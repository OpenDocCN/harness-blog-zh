无人机和 AWS 应用程序 Runner -一起更好| Harness

原文：https://www.harness.io/blog/drone-aws-app-runner#0001-01-01

开始使用 Drone 来处理构建和发布部分是轻而易举的事情，然后允许 AWS App Runner 部署到 AWS 基础架构中。

随着越来越多的开发者资源转向公共云，亚马逊网络服务最近发布了 AWS App Runner 。AWS App Runner 的前提是工程师可以只关注源代码，不需要具备底层基础设施的知识。AWS App Runner 似乎是 AWS 首次尝试提供针对 AWS 平台的平台即服务 (PaaS)。

AWS App Runner 酷似牧场主里约；通过为部署创建可公开访问的路由，消除服务的网络/路由复杂性。开始使用 Drone 来处理构建和发布部分是轻而易举的事情，然后允许 AWS App Runner 部署到 AWS 基础架构中。
